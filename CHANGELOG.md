# vue3-smooth-scrollbar

## 2.0.0

### Major Changes

- Modernize the package with Vite, TypeScript, and Bun. The component now uses `<script setup>`, ships optional `styles.css`, and targets Vue 3.5+ with `smooth-scrollbar` as a peer dependency. Removed lodash, core-js, and Vue CLI in favor of ESLint 9 and Changesets-based releases.
